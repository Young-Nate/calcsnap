Percentage Increase Formula

The percentage change between two values tells you how much something grew or shrunk relative to the original. It's used for comparing prices, salaries, stock returns, and any measurement over time.

% Change = ((New Value − Old Value) / |Old Value|) × 100 New Value = Old Value × (1 + Change% / 100)
Example — Price from €80 to €100

Change = ((100 − 80) / 80) × 100 = +25%

The price increased by 25%.

Example — Applying a 30% increase to 500

New Value = 500 × (1 + 0.30) = 500 × 1.30 = 650
