Percentage Difference Calculator: Formula, Examples, and Common Mistakes (2026)

Updated: April 14, 2026 • Reading time: ~8 minutes

“Percentage difference” sounds like “percent change,” but they solve different problems. If you use the wrong one, your results can be misleading—especially when comparing two prices, two salaries, or two budget numbers where neither is the obvious “starting point.” This guide explains the percentage difference formula, shows step-by-step examples, and helps you pick the right calculator for the job.

Quick takeaway: Use percentage difference when comparing two values without a clear “before” and “after.” Use percent change when you have a starting value and want to measure increase/decrease from that baseline.

What is percentage difference?

Percentage difference is a way to express how far apart two numbers are, relative to their average. It answers: “How different are these two values, as a percentage of a typical value between them?”

When it’s useful

  • Comparing offers: Two job salaries in the same city.
  • Shopping comparisons: Two suppliers quote different prices for the same item.
  • Budget checks: Two months of spending when neither is the “baseline.”
  • Estimates vs actuals: A forecast compared to actual results (when you want symmetry).

The percentage difference formula

The most common definition uses the average of the two values as the denominator:

Percentage difference = \( \frac{|A - B|}{(A + B)/2} \times 100\% \)

Where:

  • \(A\) and \(B\) are the two values you’re comparing
  • \(|A - B|\) is the absolute difference (always positive)
  • \((A + B)/2\) is their average

Why the average matters

Using the average makes the calculation symmetric: swapping \(A\) and \(B\) gives the same result. That’s exactly what you want when you’re comparing two values without a clear “starting” number.

Percentage difference vs percent change (don’t mix these up)

Percent change

Use when you have a baseline (a “before” value).

(New − Old) / Old × 100%

Example: price went from $80 to $100.

Percentage difference

Use when comparing two values as peers.

|A − B| / ((A + B)/2) × 100%

Example: Supplier A is $80, Supplier B is $100.

If you’re building a personal finance habit—tracking spending, comparing quotes, checking salary ranges—percentage difference is often the better “apples-to-apples” metric.

Step-by-step examples

Example 1: Comparing two prices

Supplier A quotes $80. Supplier B quotes $100.

  1. Difference: \(|80 - 100| = 20\)
  2. Average: \((80 + 100)/2 = 90\)
  3. Percentage difference: \(20/90 \times 100\% \approx 22.22\%\)

Answer: The two prices differ by about 22.22%.

Example 2: Salary comparison without a “starting salary”

Offer A: $72,000. Offer B: $78,000.

  1. Difference: \(|72,000 - 78,000| = 6,000\)
  2. Average: \((72,000 + 78,000)/2 = 75,000\)
  3. Percentage difference: \(6,000/75,000 \times 100\% = 8\%\)

Answer: The offers are about 8% apart.

Example 3: Comparing two months of spending

March spending: $2,350. April spending: $2,120.

  1. Difference: \(|2,350 - 2,120| = 230\)
  2. Average: \((2,350 + 2,120)/2 = 2,235\)
  3. Percentage difference: \(230/2,235 \times 100\% \approx 10.29\%\)

Answer: Your spending differs by about 10.29% between those months.

Common mistakes (and how to avoid them)

1) Using percent change when you meant percentage difference

If you use percent change with the “wrong” baseline, you’ll get a different number. For example, from $80 to $100 percent change is \((100-80)/80 = 25%\). From $100 to $80 it’s \((80-100)/100 = -20%\). Those aren’t symmetric. Percentage difference avoids this confusion when comparing peers.

2) Forgetting the absolute value

Percentage difference is typically reported as a positive magnitude. If you drop the absolute value, you might report a negative number, which is more like percent change.

3) Dividing by the wrong denominator

The denominator is the average \((A+B)/2\), not \(A\) or \(B\). If you divide by one value, you’ve switched to a percent-change style calculation.

4) Edge cases: zero and negative numbers

  • If \(A\) and \(B\) are both 0, the formula divides by zero and is undefined.
  • If one value is 0 and the other is non-zero, the average is non-zero, so it works (but interpret carefully).
  • If values can be negative (e.g., profit/loss), percentage difference can still be computed, but “average” in the denominator may be small, making the result very large. Consider whether absolute magnitude comparisons make sense for your context.

FAQ

Is percentage difference the same as percent error?

No. Percent error typically compares an observed value to a “true” or accepted value: \(|Observed - True| / |True| \times 100%\). Percentage difference treats both values symmetrically using the average.

Why is my percentage difference over 100%?

That can happen when the difference between \(A\) and \(B\) is larger than their average. Example: \(A=10\), \(B=40\). Difference is 30; average is 25; result is 120%.
